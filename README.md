# Sistema de Gestión Documental
Sistema web desarrollado como proyecto académico para la gestión y consulta de documentos.

## Tecnologías utilizadas
- PHP
- HTML
- CSS
- My SQL

## Funcionalidades
- Registro de usuarios
- Inicio de sesión por roles
- Carga de documentos
- Actualizacion de documentos
- Funcionalidades segun el rol del usuario
- Visualizacion de los documentos

##Rol del desarrollador
Desarrollo completo del sistema como proyecto académico, incluyendo lógica backend en PHP, estructura frontend y manejo de base de datos.

##  Propósito del proyecto
Aplicar conocimientos de desarrollo web y sistemas de información en un caso práctico real.
